# lodash.js

Lodash是一个著名的javascript原生工具库，不需要引入其他第三方依赖。
类似的库有,  underscorejs, lazyjs 等

首先要明白的是lodash的所有函数都不会在原有的数据上进行操作，而是复制出一个新的数据而不改变原有数据。类似immutable.js的理念去处理。

lodash 里面的 find , 最近的 js 也有, 但是 lodash 兼容老的浏览器
