# 代码块

代码块的优点:

可以给代码逻辑 分块

```js
const arr = [1, 2, 3, 4];
let node1 = arr.filter((e) => e < 2);
// do something;
let node2 = arr.filter((e) => e > 2);
// do something;
```

```js
const arr = [1, 2, 3, 4];

// 1. 步骤1
{
  let node = arr.filter((e) => e < 2);
  // do something;
}
// 2. 步骤2
a: {
  let node = arr.filter((e) => e > 2);
  // do something;
}

label: {
  break label; // 可以通过 break 跳出改代码块, @ switch
}
```
