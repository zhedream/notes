## import @ vscode 提示
1. 不再手写import - VSCode自动引入Vue组件和Js模块
https://www.cnblogs.com/shanjinghao/p/12357091.html

在根目录添加 jsconfig.json 
```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": [
        "./src/*"
      ]
    },
  },
  "include": [
    "./src/**/*",
  ],
}
```
