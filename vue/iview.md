## Input

'i-input': 'Input'

```html
<i-input
  v-model="value"
  placeholder="Enter something..."
  clearable
  style="width: 300px"
></i-input>
```

```html
  placeholder="请选择创建时间"
  style="width: 300px"
></Date-Picker>
```
